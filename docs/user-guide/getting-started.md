# Getting Started with InsightEngine AI

## 1. Logging In

Access the platform at `http://your-instance-url`. Log in using your organizational credentials or SSO provider.

## 2. Connecting Data

1. Navigate to **Data Sources** in the sidebar.
2. Click **Add New Source**.
3. Select your database type (Postgres, MySQL, Snowflake, etc.).
4. Enter credentials and click **test Connection**.

## 3. Exploring Data

Use the **Data Explorer** to browse tables and schemas without writing SQL. You can profile columns to see value distributions and basic statistics.

## 4. Creating Your First Chart

1. Go to **New > Chart**.
2. Select a Dataset.
3. Drag fields to **X-Axis** and **Y-Axis**.
4. Save the chart to a Dashboard.
