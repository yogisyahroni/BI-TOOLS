# Working with Dashboards

Dashboards are the core visualization layer of InsightEngine AI.

## Creating a Dashboard

1. Click **New > Dashboard**.
2. Name your dashboard and select a layout template (Grid, Freeform).
3. Click **Create**.

## Adding Cards

Dashboards consist of **Cards**. A card can be:

- A Chart (Bar, Line, Pie, etc.)
- A Pivot Table
- A Markdown Text block
- A Filter control

### To add a card

1. Click **Edit** in the top right.
2. Click **Add Card**.
3. Choose an existing chart or create a new one.

## Filters

You can add global filters that apply to all cards on the dashboard.

- **Date Range Filter**: Filters time-series data.
- **Categorical Filter**: Filters by specific column values (e.g., Region, Status).

## Sharing

Click **Share** to:

- Generate a public link (password protected).
- Invite specific team members.
- Schedule an email report (PDF/PNG).
