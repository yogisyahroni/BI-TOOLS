# Alerts

Set up alerts to get notified when your data changes.

## Creating an Alert

1. Navigate to **Alerts**.
2. Click **Create Alert**.
3. Define the condition (e.g., `Sales > 1000`).
4. Select the notification channel (Email, Slack, Teams).

## Notification Channels

- **Email**: Native support.
- **Slack**: Incoming Webhook.
- **Microsoft Teams**: Incoming Webhook.

## Managing Alerts

You can pause, edit, or delete alerts from the Alerts list view.
