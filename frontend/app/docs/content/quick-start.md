# Quick Start Guide

Welcome to InsightEngine! This guide will help you set up your account, connect your first data source, and create your first dashboard.

## 1. Create an Account

If you haven't already, sign up for an account. You'll need to verify your email address before you can log in.

## 2. Connect a Data Source

InsightEngine connects to your existing databases. We support:

- PostgreSQL
- MySQL
- SQLite
- DuckDB

To connect a source:

1. Navigate to **Settings > Data Sources**.
2. Click **Add New Source**.
3. Select your database type and enter connection details.
4. Click **Test Connection** to verify.

## 3. Create a Dashboard

Once your data is connected, you can start visualizing it.

1. Go to **Dashboards**.
2. Click **New Dashboard**.
3. Use the **Visual Query Builder** to select data and choose a chart type.
4. Drag and drop charts to arrange your layout.
