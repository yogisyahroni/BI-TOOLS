# Data Sources

InsightEngine supports a variety of data sources.

## Supported Databases

- **PostgreSQL**: Full support for tables, views, and materialized views.
- **MySQL**: Standard connection.
- **SQLite**: Local file support.
- **DuckDB**: High-performance analytical database.

## Adding a Source

1. Go to **Settings > Data Sources**.
2. Click **Add Source**.
3. Fill in the connection details (Host, Port, User, Password, Database).
4. Click **Save**.

## Connection Status

The list will show the status of each connection:

- **Green**: Connected and healthy.
- **Red**: Error connecting. Check logs for details.
